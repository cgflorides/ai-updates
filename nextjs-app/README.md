# Next.js Version

This is the more structured version of the AI Updates app.

## Why this version exists

- it is easier to grow than a plain static website
- it gives you a more professional project structure
- it is a better base for future features like AI summaries, saved stories, and login

## Folder overview

- `app/` contains the main pages and global styles
- `components/` contains reusable UI parts
- `lib/` contains the news-fetching logic

## How to run it later

Once `node` and `npm` are installed:

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.
